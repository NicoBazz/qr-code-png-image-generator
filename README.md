# 🔗 QR Code PNG Image Generator

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

A simple Python script that generates QR codes from any text string and saves them as PNG images.

---

## 🚀 Features

- Generate a QR code from any string (URL, text, etc.)
- Output saved as a PNG image file
- Lightweight and easy to use

---

## 📋 Requirements

```
qrcode
Pillow
```

Install dependencies:

```bash
pip install qrcode[pil]
```

---

## 💻 Usage

```python
python qr.py
```

Just edit the string inside the script and run it — a QR code PNG will be generated in the same folder.
